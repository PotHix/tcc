Canvas

Canvas é uma API que permite ao desenvolvedor desenhar objetos na tela utilizando uma forma pré-determinada interface.

Com canvas é possível criar diversas formas geométricas, importar imagens, criar curvas e etc.

Grande parte dos navegadores já possuem suporte a canvas e a cada nova versão o suporte é melhorado, tornando as interações mais rápidas.

DOM e transformações CSS3

Também é possível desenvolver jogos utilizando DOM seguindo a mesma linha de estruturação das páginas de internet comuns.

Ao utilizar DOM não é necessário o redesenho de toda a área bastando apenas o reposicionamento dos elementos.

Utilizando DOM se torna possível a aplicação de transições e outros efeitos CSS3, que permitem efeitos acelerados por hardware para o jogo.

SVG

SVG é uma especificação que permite a utilização de gráficos vetoriais diretamente no navegador.

Utilizando essa especificação é possível fazer animações diretamente no navegador com suporte nativo.

Websockets

A tecnologia de Websockets traz o poder dos sockets do desktop para o navegador.

Com essa tecnologia é possível abrir uma conexão com o servidor e trocar dados de uma maneira mais rápida.

Essa tecnologia é muito importante para jogos com mais de um jogador, pois ela fará a conexão entre computadores remotos.

Offline cache

Essa tecnologia define uma maneira de deixar uma página de internet disponível mesmo quando o usuário não possui acesso a internet.

Utilizando essa tecnologia é possível criar jogos casuais que podem ser jogados em qualquer lugar.

Local storage

Com Local storage é possível guardar informações no navegador do usuário para consultas posteriores.

Essa tecnologia vem para acabar com a utilização de cookies para guardar informações da aplicação.

Utilizando-a será possível guardar informações especícicas sobre o jogo sem precisar de acesso ao dispositivo do jogador.

Audio

O HTML5 possui em sua especificação uma implementação para tratar audio nativamente no navegador e vários navegadores já possuem essa implementação.

Juntamente com a implementação de Audio existe também a implementação chamada WebAudioApi, que permite sons tem tempo real para o navegador.

Essas duas tecnologias podem prover toda a sonosplastia necessária para um jogo casual simples.

WebGL

A tecnologia de WebGL provê aceleração 3D via navegador.

Utilizando-a será possível conversar diretamente com a GPU do dispositivo para fazer calculos 3D utilizando OpenGL ES.

Essa tecnologia está ganhado espaço e cada vez mais conseguindo mais suporte, e com ela será possível melhorar cada vez mais os jogos casuais.

  • Jogos simples podem funcionar tanto em computadores como em celulares.
  • HTML5 é uma tecnologia plausível de utilização para jogos casuais.
